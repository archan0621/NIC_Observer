# ServerSpell

Simple script to monitoring status of network interfaces in linux

## Getting Started

리눅스의 네트워크 인터페이스의 상태를 지속적으로 모니터링하는 스크립트

### How to Build & Start 

아래의 명령어로 프로젝트를 실행할 수 있습니다.

```
sh NIC_Observer.sh
```

## Execute With

* [Bash Shell](https://www.gnu.org/software/bash/) - 실행용 쉘(기타 쉘에서도 실행 가능)
